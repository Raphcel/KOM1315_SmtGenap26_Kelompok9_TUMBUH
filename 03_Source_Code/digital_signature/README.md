# Digital Signature

Folder ini digunakan untuk implementasi tanda tangan digital pada TUMBUH. Bagian ini mendukung kebutuhan integritas data dan non-repudiation pada aksi penting.

## Cakupan

- Pembuatan pasangan kunci untuk kebutuhan pengujian.
- Proses signing pada transaksi atau perubahan data penting.
- Proses verification untuk memastikan data tidak berubah.
- Integrasi digital signature dengan workflow utama aplikasi.
- Dokumentasi cara private key dilindungi.

## Contoh Aksi yang Dapat Ditandatangani

- Perubahan status lamaran oleh HR.
- Persetujuan atau penolakan data penting oleh admin.
- Perubahan data yang membutuhkan bukti integritas.
