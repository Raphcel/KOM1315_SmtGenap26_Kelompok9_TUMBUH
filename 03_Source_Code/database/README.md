# Database Security

Folder ini digunakan untuk artefak keamanan yang berhubungan dengan database TUMBUH.

## Cakupan

- Identifikasi tabel dan kolom sensitif.
- Rancangan hashing dan salting untuk password.
- Rancangan enkripsi untuk data yang membutuhkan perlindungan tambahan.
- Skema database atau migrasi yang terkait dengan keamanan.
- Contoh query atau dokumentasi validasi keamanan data.

## Bukti yang Diharapkan

- Catatan kolom mana yang dianggap sensitif dan alasannya.
- Dokumentasi cara password disimpan secara aman.
- Dokumentasi cara data terenkripsi disimpan dan diakses.
- Migrasi atau skema yang sudah disanitasi jika diperlukan.
