# 04 Reports & Paper

Folder ini berisi luaran laporan monitoring, laporan akhir, user manual, dan paper ilmiah untuk Proyek PBL Keamanan Informasi.

## Struktur

| Folder | Keterangan |
|---|---|
| `Monitoring_P7/` | Laporan kemajuan tahap monitoring, terutama Bab 1-3 dan bukti demo |
| `Final_Technical_Report/` | Laporan teknis akhir, dokumen kompilasi, dan user manual |
| `Scientific_Paper/` | Paper ilmiah format JIKA dan bukti submit/LoA jika tersedia |

## Catatan

Isi laporan harus konsisten dengan dokumen proposal, threat modeling, rancangan, source code, dan hasil pengujian. Seluruh artefak akhir perlu tersedia sebelum tenggat akhir pengumpulan.
