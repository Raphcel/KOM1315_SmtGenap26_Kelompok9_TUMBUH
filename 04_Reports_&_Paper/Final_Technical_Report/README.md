# Final Technical Report

Folder ini digunakan untuk laporan teknis akhir proyek TUMBUH pada mata kuliah Keamanan Informasi.

## Isi yang Diharapkan

- Laporan teknis final.
- Dokumen kompilasi hasil analisis, rancangan, implementasi, dan pengujian.
- User manual atau panduan penggunaan sistem.
- Panduan instalasi atau deployment jika diperlukan.
- Ringkasan implementasi AAA Security.
- Ringkasan hasil pengujian keamanan.

## Catatan

Laporan akhir harus menunjukkan hubungan yang jelas antara ancaman yang ditemukan, kebutuhan keamanan, implementasi yang dibuat, dan hasil pengujiannya.
