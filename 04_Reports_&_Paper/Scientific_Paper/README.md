# Scientific Paper

Folder ini digunakan untuk luaran paper ilmiah proyek PBL.

## Isi yang Diharapkan

| Artefak | Status | File |
|---|---|---|
| Draft paper ilmiah sesuai format JIKA | Tersedia | `jurnal_tumbuh.tex` |
| Versi PDF paper | Tersedia | `jurnal_tumbuh.pdf` |
| Bukti submit paper | Belum tersedia | Tambahkan screenshot/status submission jika sudah submit |
| Bukti LoA | Belum tersedia | Tambahkan Letter of Acceptance jika diterima |
| Revisi paper | Belum ada | Tambahkan jika ada masukan dosen/reviewer |

## Catatan

Paper perlu merangkum latar belakang masalah, kelemahan sistem awal, metode integrasi keamanan, implementasi AAA Security, hasil pengujian, dan kesimpulan.

Status publikasi saat ini: **Draft Internal**. Jangan mengklaim status
`Submitted`, `Under Review`, atau `Accepted` sebelum ada bukti submission,
status review, atau LoA.
