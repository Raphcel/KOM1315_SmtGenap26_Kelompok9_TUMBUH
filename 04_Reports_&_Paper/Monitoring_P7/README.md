# Monitoring P7

Folder ini digunakan untuk luaran monitoring pada pertemuan atau minggu ke-7.

## Isi yang Diharapkan

- Laporan kemajuan Bab 1-3.
- Ringkasan progres analisis dan perancangan keamanan.
- Bukti demo atau tautan video demo jika diminta.
- Screenshot progres aplikasi atau modul keamanan.
- Catatan pembagian tugas anggota kelompok jika diperlukan.

## Catatan

Dokumen monitoring menjadi bukti bahwa kelompok sudah melakukan retroactive upload dan memiliki baseline proyek sebelum melanjutkan implementasi keamanan pada minggu berikutnya.
